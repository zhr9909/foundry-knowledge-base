## 2. Layout Chunker

`scripts/layout_chunker.py`

### 切块策略

- 按 layout SectionHeader/Title 标签确定章节边界
- 跨页表格自动合并（同列数 + 无新标题打断）
- 同一章节内的 Text/List 合并为一个 text chunk
- 表格独立 chunk（保留 markdown 格式）

### Metadata 字段

```json
{
  "section_path": "390 / Wrought Aluminum Alloy",
  "page_range": [236],
  "material_tags": ["6061", "aluminum", "t6"],
  "source": "ASM Handbook Volume 2B: Aluminum Alloys"
}
```

### section_path 提取来源

```
布局文件 detected SectionHeader bbox → 从 PDF 原文截取标题文字
  page.get_text("text", clip=Rect(x0*sx, y0*sy, x1*sx, y1*sy))
  → "390 / Wrought Aluminum Alloy"
```
